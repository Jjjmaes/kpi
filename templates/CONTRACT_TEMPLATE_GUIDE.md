# 合同模板创建指南

## 快速开始

1. 使用 Microsoft Word 创建一个新的 `.docx` 文档
2. 按照你的合同格式要求编写合同内容
3. 在需要插入动态数据的位置，使用 `{变量名}` 语法
4. 保存文件为 `contract-template.docx` 并放置在 `templates/` 目录下

## 项目成员列表（可选）

如果需要显示项目成员列表，可以在适当位置添加：

```
{#if hasMembers}
项目成员：
{#each members}
• {role}：{name}（{username}）
{/each}
{/if}
```

## 重要提示

1. **变量语法**：使用 `{变量名}` 格式，注意花括号必须是英文半角
2. **条件判断**：使用 `{#if 变量}内容{/if}` 格式
3. **循环**：使用 `{#each 数组}内容{/each}` 格式
4. **文件位置**：模板文件必须命名为 `contract-template.docx` 并放在 `templates/` 目录
5. **格式保持**：Word 中的格式（字体、字号、颜色、对齐等）会被保留
6. **测试**：创建模板后，建议先用一个测试项目导出合同，检查格式和变量替换是否正确

## 常见问题

**Q: 变量没有替换？**
A: 检查变量名是否正确，注意大小写，确保使用英文半角花括号。

**Q: 条件判断不工作？**
A: 确保 `{#if}` 和 `{/if}` 正确配对，布尔变量使用 `{#if 变量}` 而不是 `{#if 变量 == true}`。

**Q: 循环不显示？**
A: 确保数组变量名称正确，使用 `{#each members}` 和 `{/each}` 正确配对。

**Q: 格式丢失？**
A: docxtemplater 会尽量保留 Word 格式，但某些复杂格式可能需要调整。
